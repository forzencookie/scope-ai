# Förening — Medlemmar, Stämma & Skatt

## Två föreningsformer
- **Ideell förening:** icke-vinstdrivande syfte (idrott, kultur, välgörenhet). Ofta skattebefriad på verksamhetsinkomster om föreningen är allmännyttig.
- **Ekonomisk förening:** främjar medlemmarnas ekonomiska intressen, minst 3 medlemmar, kan dela ut överskott.

## Medlemsregister (obligatoriskt)
Ingen aktiebok — föreningen för ett **medlemsregister** med namn, adress och inträdesdag per medlem. Demokratisk princip: **en röst per medlem** oavsett kapitalinsats.

## Avgifter
Medlemsavgifter är normalt inte skattepliktiga för ideella föreningar.

## Årsmöte
Hålls inom **6 månader** efter räkenskapsårets slut (inte bolagsstämma).
Obligatoriska punkter:
- Fastställa verksamhetsberättelse
- Fastställa resultaträkning och balansräkning
- Beslut om resultatdisposition
- Ansvarsfrihet för styrelsen
- Val av styrelse och revisorer

## Styrelsens ansvar
Styrelseledamöter kan bli personligt ansvariga om föreningen inte uppfyller sina förpliktelser (t.ex. skatteskulder, leverantörsskulder).
